# GitHub setup for Ezio skills and connectors

Use this repository as the source of truth for custom OpenClaw skills and integration notes.

## Suggested structure

- `skills/` custom skills
- `connectors/` connector-specific notes, schemas, or helpers
- `docs/` project notes if needed

## Starter skills created

- `skills/ar-pay-blog-workflow`
- `skills/social-creative-brief`
- `skills/ai-video-reference-pipeline`

## Recommended next steps

1. Create a GitHub repo.
2. Push this workspace or copy the `skills/` folder into that repo.
3. Keep secrets out of git.
4. Iterate on skills from real use.

## How to use with OpenClaw

Options:
- Keep the repo cloned in the workspace and edit skills there.
- Or copy finished skills into another shared skills location.
