# Workflow notes

## Browser-first workflow

Use the normal logged-in browser profile on this machine.
If Freepik is no longer logged in, stop and ask Mitko to log in again.

## Standard Mitko workflow
- generate image first
- choose the strongest image result
- reuse that image as a reference for video
- preserve character and environment consistency
- download the final chosen outputs

## Search checklist
- identify subject
- identify style
- identify intended use
- shortlist strong matches

## AI image checklist
- lock subject
- lock style
- lock framing
- note prompt changes between versions
- choose the best reference candidate for later video work

## AI video checklist
- start from the chosen image reference when possible
- define fixed traits
- define allowed motion/scene changes
- preserve character and environment consistency
- log the final prompt used

## Download checklist
- confirm final chosen output
- confirm destination if downloading locally
- report saved path back to Mitko
