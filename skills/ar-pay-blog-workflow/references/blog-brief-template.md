# Blog Brief Template

Use this structure when the user asks for a blog brief.

## Core brief
- Title / working title
- Goal
- Audience
- Funnel stage
- Primary CTA
- Supporting CTA
- Product / offer / campaign
- Publish deadline

## Search intent
- Primary keyword/theme
- Secondary keyword/theme
- User intent
- Questions to answer

## Article structure
- H1 options
- Intro angle
- Section outline
- Proof points or examples needed
- CTA placement

## Delivery extras
- Meta description
- Short social caption
- Visual ideas
