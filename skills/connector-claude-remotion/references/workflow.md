# Workflow

## Claude side

Use Claude for:
- video ideas
- scene outlines
- CTA options
- script compression
- pacing choices
- alternate hooks

## Remotion side

Use Remotion for:
- vertical 9:16 compositions
- timed text layers
- background assets
- transitions
- end cards
- exports

## Good pattern

- Scene 1: hook
- Scene 2: context
- Scene 3: proof or mechanic
- Scene 4: CTA

Keep each scene modular so edits do not break the full render.
