# Social Creative Brief

Translate loose ideas into production-ready creative direction.

## Workflow

1. Identify platform and format.
2. Identify audience and campaign goal.
3. Identify the core message.
4. Define visual direction.
5. Define copy hierarchy.
6. Add production notes.

## Capture the minimum brief

Collect or infer:
- platform
- format
- aspect ratio
- audience
- offer or message
- tone
- CTA
- deadline
- required brand elements

## Build the brief

Include:
- objective
- audience
- single main message
- hook options
- visual concept
- layout guidance
- copy hierarchy
- CTA options
- deliverables
- production notes

## Visual direction rules

Specify only the details that matter for execution:
- composition
- subject
- color mood
- typography feel
- key objects or symbols
- do-not-use notes

Prefer concise visual direction over bloated art direction paragraphs.

## Output formats

Return one of:
- short creative brief
- detailed creative brief
- hook pack
- copy hierarchy
- art direction notes
- revision notes
